# Startup Success Analyzer

## Overview
This project analyzes startup success using statistical techniques in R.

## Objective
To identify key factors influencing startup growth and check if sectors behave differently.

## Techniques Used
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Multivariate Normality Test (Mardia)
- Principal Component Analysis (PCA)
- ANOVA

## Key Insights
- Startup scale does not significantly differ across sectors
- Growth is not strongly industry-dependent
- PCA successfully reduced multiple features into meaningful components

## Tools
- R
- ggplot2
- MVN
- factoextra
